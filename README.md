该Hbuilderx插件使用 https://github.com/sainoba/vscode-px-to-rem修改

thanks @sainoba

主要配置
```
"keybindings": [
    {
        "command": "extension.pxToremAndRemToPx",
        "key": "Alt+z"
    },
    {
        "command": "extension.pxPerRem",
        "key": "Alt+s"
    }
]
```
